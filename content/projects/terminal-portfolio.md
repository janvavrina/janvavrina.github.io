# Terminal Portfolio

This is the project you're currently viewing!

## Overview

A personal portfolio website that mimics a terminal interface. Users can navigate using familiar commands like `ls`, `cat`, and `cd`.

## Features

- **Terminal UI**: Authentic terminal look and feel
- **Catppuccin Theme**: Beautiful Mocha color scheme
- **Markdown Rendering**: Content is displayed beautifully
- **Command History**: Use arrow keys to navigate history
- **Easter Eggs**: Try `neofetch` or `sudo`!

## Tech Stack

- TypeScript
- Vite
- marked.js for Markdown
- Pure CSS with Catppuccin colors
